Что такое синхронизация?

Синхронизация загружает актуальные данные о ЖК и квартирах из официального API BI Group. Происходит обновление цен, доступности квартир и характеристик объектов.

AI Анализ

При синхронизации каждый ЖК анализируется AI для определения берега (Левый/Правый), атмосферы района и автоматического тегирования. Это улучшает качество поиска.

Частота обновления

Рекомендуется запускать синхронизацию 1-2 раза в день для актуальности данных. Процесс занимает 1-2 минуты в зависимости от количества объектов.
